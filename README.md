# Pi Showroom Indonesia

Perusahaan kami berdiri  dengan nama lengkap Cv. Pi Showroom Indonesia Hackathon adalah platform situs web perdagangan bergerak  Pengadaan Barang dan Jasa. Awalnya, perusahaan kami hanya melayani pasar lokal dengan fokus pada pengadaan barang untuk sektor industri kecil dan menengah. Namun, dengan komitmen untuk memberikan layanan berkualitas dan kepuasan pelanggan, kami berhasil tumbuh pesat dan memperluas jangkauan layanan kami.
kami secara progresif mengembangkan portofolio layanan kami untuk mencakup berbagai sektor industri, seperti (Kendaraan, Properti, Elektronik, perhiasan, bahan pokok pangan, Agribisnis, handicraft serta Fashion). Kini, kami telah menjadi pemain yang diakui dalam industri pengadaan barang dan jasa, beroperasi tidak hanya di tingkat nasional, tetapi juga telah berhasil melakukan ekspansi ke pasar internasional.
